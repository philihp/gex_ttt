## Running

    $ git clone https://github.com/philihp/gex.git
    $ git clone https://github.com/philihp/gex_ttt.git
    $ cd gex_ttt
    $ elixir -S mix run -e GexTtt.start

* Boilerplate example game
* Minimax search
